This CSV contains model analysis on different association types.

We sampled ~1K cue-image pairs of the instances created via the game with 10-12 candidates for analysis. Three annotators were instructed to classify the associations to six categories: (a) visually salient; (b) visually non-salient; (c) concept-related; (d) activity; (e) counting; (f) colors. Full description in Section 3.3 in the paper. Annotation guidelines are available here.
